What's New in Deezer BPM

v0.6.4
🐛 Bug Fixes v0.6.4
🎯

Editing a BPM after reordering now hits the right track

After moving a song to a new position in a playlist, editing its BPM could update the track that was previously in that spot. Edits now always apply to the track shown in the row.

✏️ New Features v0.6.0
✏️

Edit any BPM by double-clicking New

Double-click any BPM badge in a playlist or queue to enter a value manually. Works on tracks showing N/A as well as tracks that already have an API-provided BPM. Your value is saved and persists across sessions.

↩️

Revert to the original API value

If you entered a manual BPM and want to go back, double-click the badge, clear the input, and press Enter. The original value downloaded from Deezer's API will reappear immediately.

⬡

Visual indicator for manual values

Manually entered BPM badges use a dashed border 128 so you can tell at a glance which values came from you versus the API.
